# 外设行业报告网站

这是一个专注于外设行业的专业报告网站，提供市场分析、产品评测、趋势预测等内容。

## 网站结构

- **首页**: 行业概览和最新报告
- **市场分析**: 各细分市场的深度分析
- **产品评测**: 键盘、鼠标、显示器等外设产品评测
- **趋势预测**: 行业发展趋势和技术创新
- **数据统计**: 市场份额、销售数据等统计信息
- **关于我们**: 团队介绍和联系方式

## 技术栈

- **静态站点生成器**: Jekyll
- **托管平台**: GitHub Pages
- **样式框架**: Bootstrap 5
- **图表库**: Chart.js
- **部署**: 自动化CI/CD

## 快速开始

1. Fork此仓库
2. 在本地安装Jekyll: `gem install jekyll bundler`
3. 安装依赖: `bundle install`
4. 本地预览: `bundle exec jekyll serve`
5. 部署到GitHub Pages

## 内容贡献

报告内容位于 `_posts/` 目录，按照日期命名。
数据文件位于 `_data/` 目录，支持YAML和JSON格式。
